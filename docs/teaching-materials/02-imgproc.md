# Practical Image Processing

Complete guide to filtering, morphology, thresholding, and color space operations.

[Download PDF]({{ site.baseurl }}/teaching_materials/02-image-processing.pdf){: .btn .btn-primary .fs-5 .mb-4 .mb-md-0 }

## Topics Covered

- **Filtering** - Blur, Gaussian, median, bilateral filters
- **Morphology** - Erosion, dilation, opening, closing
- **Thresholding** - Binary, adaptive, Otsu methods
- **Color Spaces** - BGR, HSV, grayscale conversions
- **Edge Detection** - Canny, Sobel, Laplacian
- **Contours** - Finding, drawing, and analyzing contours
